# Searchuse
A turn-based multiplayer strategy game where you search for and use various items.
Two words, endless possibilities.

<img width="426" height="240" alt="showcaseGif" src="https://github.com/user-attachments/assets/94bff7f7-ccc9-40c0-acdd-65a3b8586470" />

Download & try here:

https://ai-zamden.itch.io/searchuse

To launch:
- Press download
- In files, locate the zip, right click, and press extract all
- Find the folder you extracted to, and click "Searchuse 2.exe"
- Play bot or play a friend using the join code

What Searchuse is all about
- Move, Search and Use
- Try to locate and attack the enemy before he does it to you
- Intel is key
- Use strategic thinking to create plans
- Use logical deduction/reasoning to infer deeper layers of information about the enemy
- Don't step on landmines!!
